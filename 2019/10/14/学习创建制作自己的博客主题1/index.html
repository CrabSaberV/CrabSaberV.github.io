学习创建制作自己的博客主题<一>

发表于 2019-10-14更新于 2019-10-18阅读次数

前言

最近开始了学习搭建博客的道路，但是在网上找了许多主题，都不是非常满意（也许是我眼光问题），但我自身前端开发知识欠缺，因此，给自己下了大的决心，打算自己为自己设计一款好的博客主题。

而目前我所使用的主题是在hexo主题中的stun主题，感谢这位大佬的分享，个人比较喜欢，但是也有一些功能不尽人意。那么总不能一味的靠别人喂给自己，我也是时候成长啦。接下来就要开始我主题设计之旅了。

开始

首先需要安装好hexo，（后面再加入相关教程 或 链接）
…相关东西安装好后

首先自己在某个适合的位置创建一个文件夹作为我们的工作站（我就建立一在F:\GitBench\MyHexoThem）
在创建好的文档下打开shall （这里我所在环境是windows ，因此使用的power shall，文件夹上面点击文件就能找到）

输入以下命令 进行初始化
hexo init
提示一下信息表示成功(出现 Start blogging with Hexo! 即可)
